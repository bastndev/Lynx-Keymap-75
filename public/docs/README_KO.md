> [!IMPORTANT]
>
> **`백틱` = ( ` )**은 모든 키보드 레이아웃을 지원합니다 🌎
>
> ```ts
> exports.BACKTICK_EQUIVALENTS = [
>   { Backquote: "`" }, // 영어, 미국
>   { Backquote: "º" }, // 스페인어 (스페인)
>   { Backquote: "'" }, // 프랑스어, 독일어
>   { Backquote: "²" }, // 프랑스어
>   { Backquote: "^" }, // 독일어, 스위스
>   { Backquote: "ё" }, // 러시아어
>   { Backquote: "~" }, // 포르투갈어, 이탈리아어
>   { Backquote: "半角/全角" }, // 일본어
>   { Backquote: "|" }, // 스페인어 (라틴 아메리카)
>   { Backquote: "¬" }, // 영국, 라틴 아메리카
>   { Backquote: "¨" }, // 독일어, 스웨덴어
>   { Backquote: "´" }, // 포르투갈어 (라틴 아메리카)
>   { Backquote: "₺" }, // 터키어
>   { Backquote: "₤" }, // 이탈리아어
>   { Backquote: "" }, // 기타...
> ];
> ```

## 📦 설치

### 방법 1 — 빠른 열기

운영 체제에 맞는 명령 팔레트를 엽니다:

- 🐧 Linux · 🟦 Windows → `Ctrl + P`
- 🍎 macOS → `⌘ + P`

붙여넣고 `Enter`를 누릅니다:

```
ext install bastndev.lynx-keymap-75
```

### 방법 2 — 확장 프로그램 보기

1. 확장 프로그램 열기 (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. **"Lynx Keymap 75%"** 검색
3. **설치** 클릭
