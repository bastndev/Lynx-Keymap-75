> [!IMPORTANT]
>
> **`Dấu huyền` (Backtick) = ( ` )** hỗ trợ tất cả các bố cục bàn phím 🌎
>
> ```ts
> exports.BACKTICK_EQUIVALENTS = [
>   { Backquote: "`" }, // Tiếng Anh, US
>   { Backquote: "º" }, // Tiếng Tây Ban Nha (Spain)
>   { Backquote: "'" }, // Tiếng Pháp, Tiếng Đức
>   { Backquote: "²" }, // Tiếng Pháp
>   { Backquote: "^" }, // Tiếng Đức, Thụy Sĩ
>   { Backquote: "ё" }, // Tiếng Nga
>   { Backquote: "~" }, // Tiếng Bồ Đào Nha, Ý
>   { Backquote: "半角/全角" }, // Tiếng Nhật
>   { Backquote: "|" }, // Tiếng Tây Ban Nha (Latin America)
>   { Backquote: "¬" }, // Anh, Latin America
>   { Backquote: "¨" }, // Tiếng Đức, Thụy Điển
>   { Backquote: "´" }, // Tiếng Bồ Đào Nha (Latin America)
>   { Backquote: "₺" }, // Tiếng Thổ Nhĩ Kỳ
>   { Backquote: "₤" }, // Tiếng Ý
>   { Backquote: "" }, // Và NHIỀU HƠN NỮA...
> ];
> ```

## ➤ Cài đặt

### Phương pháp 1 — Mở Nhanh

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Dán lệnh sau và nhấn `Enter`:

```
ext install bastndev.lynx-keymap-75
```

### Phương pháp 2 — Xem Tiện ích

1. Mở Tiện ích mở rộng (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Tìm kiếm **"Lynx Keymap 75%"**
3. Nhấp vào **Cài đặt**
